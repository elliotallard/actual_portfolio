React App: Phoenix Memoir


phoenix-memoir.netlify.com





Quickly learning React through online tutorials and peers, I created a simple react app capturing some of the more personal moments I had in Phoenix, Arizona. Building up from hand-drawn mockups to high-fidelity prototypes, the design reflects (my perceptions of) a Phoenix sensibility.

Through this project, I learned the fundamentals of React: what it is capable of (eliminating redundant html css code, integrating backends in a snap, making directory structures a whole lot more readable), and what it struggles with (having multiple states, making one-off css displays). This was the perfect way to not only get my hands dirty with React, but also archive a summer.

for the class Software Structures for Usable Interfaces
taught by Jason Hong & Michael Xieyang Liu
